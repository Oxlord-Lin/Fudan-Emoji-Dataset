This repository contains codes and raw data for the paper "Fudan Emoji Dataset (FED): Emotional and Semantic Norms of 359 Emojis in Chinese Context and Cross-Cultural Comparison". This README document provides an overview of the structure and contents of the repository.

The repository is organized into the following directories:
- "S1_dataset_aggregation_and_LLM_prompt": codes to aggregate the initial stimulus set and to interact with LLMs
- "S2_pilot_test": codes to generate questionnaires for pilot test and to analyze the results of pilot test
- "S3_stimulus_materials_for_norming_survey": codes to generate questionnaires for norming survey
- "S4_corpus_freq_count": codes to download CCI2.0 dataset and to count the frequency of emojis in the dataset.
- "S5_data_analysis": data analysis codes after the norming survey where most figures and tables in paper are generated by these codes.

## S1_dataset_aggregation_and_LLM_prompt
This directory has the following structure
``` txt
├── S1_dataset_aggregation_and_LLM_prompt/
│   ├── weibo_emoji_frequency.xlsx # Weibo emoji frequency dataset
│   ├── MLE_union_KK.xlsx # The union of MLE and Kimura and Katsurai (2017) 
│   ├── MLE_union_KK_union_weibo.xlsx # The union of MLE, Kimura and Katsurai (2017) and Weibo emoji frequency dataset
│   ├── FED.xlsx # FED dataset, it is contained here only for comparison purpose
│   ├── s1_interact_with_ernie_bot.py # codes to interact with ernie bot
│   ├── s2_seperate_examples.py # codes to separate individual sentences from the structured output of Ernie Bot
│   ├── s3_union.py # codes to union the three datasets
│   ├── s4_calculate_overlap_emoji.py # codes to calculate the number of overlapped emojis among the these datasets
```

## S2_pilot_test
This directory has the following structure
``` txt
├── S2_pilot_test/
│   ├── questionnaires/ # All the questionnaires for pilot test.
│   ├── raw_result_of_pilot_test/ # raw result of pilot test collected from participants.
│   ├── preprocessed_result_of_pilot_test/ # preprocessed result of pilot test. These are intermediate files.
│   ├── introduction.txt # The introduction of the questionnaires.
│   ├── template.txt # The template to be filled by specific emojis.
│   ├── s1_templates_filling_helper.py # generate questionnaires
│   ├── s2_basic_info_extraction.py # extract basic information from the raw result of pilot test
│   ├── s3_data_preprocessing.py # preprocess the raw result of pilot test
│   ├── s4_format_cleaning.py # clean the data
│   ├── s5_ICC_for_pilot_test.py # calculate ICC of pilot test. The ICC results can also be found in S.7 in supplementary materials of the paper.
│   ├── aggregated_preprocessed_results_of_pilot_test.xlsx # intermediate results of pilot test. This file mainly contains the candiate emotion and meaning collected from participants. It corresponds to S.5 in supplementary materials of the paper.
│   ├── emoji_set_after_manual_curation.xlsx # The candiate emotion or meaning reviewed by researchers. It corresponds to S.6 in supplementary materials of the paper.
│   ├── ICC_result.xlsx # The ICC results of pilot test. It corresponds to S.7 in supplementary materials of the paper.
```

## S5_data_analysis
This directory has the following structure
``` txt
├── S5_data_analysis/
│   ├── raw_questionaire_data/ # raw questionnaire data collected from participants in norming survey.
│   ├── emoji_image/ # all the emoji images of IOS format used in our paper. For copyright issue, we cannot provide them online.
│   ├── previous_dataset/ # previous dataset to be compared with our FED dataset.
│   ├── seperate_data_of_each_emoji_raw/ # separate data of each emoji. These are intermediate files. To save space, we do not upload them online. But they can be created by running the codes in this directory.
│   ├── seperate_data_of_each_emoji/ # separate data of each emoji. These are intermediate files. To save space, we do not upload them online. But they can be created by running the codes in this directory.
│   ├── gender_comparison/ # intermediate files for gender comparison. To save space, we do not upload them online. But they can be created by running the codes in this directory.
│   ├── emoji_unicode.txt # the unicode definition of each emoji downloaded from https://www.unicode.org/emoji/charts/full-emoji-list.html
│   ├── additional_experiment_familarity_score_raw_result.xlsx # subjective familiarity scores of the 32 participants after norming survey.
│   ├── dataset_for_experiment_with_supplementary_descriptions_by_participants.xlsx # for each emoji, participants can provide supplementary descriptions of the emojis. And this file collects the supplementary descriptions of each emoji.
│   ├── images/ # all the images generated by our codes. To save space, we do not upload them online. But they can be created by running the codes in this directory.
│   ├── FED.xlsx # the final FED dataset.
│   ├── s1_basic_info_extraction_of_raw_data.py # extract basic information from the raw questionnaire data, e.g., age, gender, etc.
│   ├── s2_create_FED_table_and_extract_unicode_definition.py # create the initial FED and extract the unicode definition of each emoji.
│   ├── s3_seperate_emoji_from_questionaire.py # seperate each emoji's data from the questionnaire results. This is for the purpose of easier data analysis.
│   ├── s4_calculate_ICC.py # calculate ICC. It corresponds to S.8 in supplementary materials of the paper.
│   ├── s5_dropout_outlier.py # drop out the outliers identified using the 3-standard-deviation criterion. It also generates Table 1 in the paper.
│   ├── s6_gender_t_test.py # gender comparison using t-test. It corresponds to S.9 in supplementary materials of the paper.
│   ├── s7_calculate_statistic.py # calculate the statictics, e.g., mean and standard deviation of each dimension, the uncertainty metrics.
│   ├── s8_descriptive_M_SD_density.py # to generate Table 2 and Figure 2 in the paper
│   ├── s9_quadratic_regression.R # to generate Figure 4 in the paper. It shows the U-shape relationship between arousal and valence.
│   ├── s10_visualization_of_embeddings.py # to generate Figure 3 in the paper. It shows the distribution of emojis in the embedding space.
│   ├── s11_correlation_analysis.py # calculate the spearman correlation among dimensions, and visualize the PCA results of emotional dimensions. It generate Figure 5, 6, 7 in the paper.
│   ├── s12_cross_culture_comparison.py # cross-cultural comparison. Generate Table 3,4,and 5 in the paper.
│   ├── s13_freq_comparison.py # Compare the frequency of emojis between FED and western dataset. It corresponds S.10 in supplementary materials of the paper.
│   ├── s14_familiarity_freq_comparison.py # Show the relationship between familiarity and frequency of emojis. It corresponds S.10 in supplementary materials of the paper.
│   ├── s15_report_raters_num_of_each_emoji.py # calculate the number of raters for each emoji.
│   ├── s16_valence_arousal_comparison_across_dataset.py # Visualize the correlation on arousla and valence among FED and other datasets. It generate the figures in Appendix A.2
│   ├── utils.py # helper functions for data analysis
```
